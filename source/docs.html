Build and Deploy SDAP locally

Guides to build and deploy SDAP using Docker are available at Read the Docs.

Deployment to Production

Included with the SDAP release (Nexus) are the requisite Helm Charts to deploy SDAP to a Kubernetes cluster (currently tested between k8s versions 1.22 and 1.25). This allows for a scalable, production deployment of SDAP to either a privately maintained cluster or to a cloud service provider like Amazon Web Service's Elastic Kubernetes Service.
